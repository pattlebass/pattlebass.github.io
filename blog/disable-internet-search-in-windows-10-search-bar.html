How to disable web search results in the start menu

Windows 10 1809 and up

Written on December 12th, 2021


Open regedit

Press Windows + R and then type regedit and hit enter. Accept the UAC prompt that shows up.
Navigate to HKEY_CURRENT_USER\SOFTWARE\Policies\Microsoft\Windows\. If there is no key named Explorer create right-click and create one. Then select it and on the right create a new DWORD value (right-click > New > DWORD (32 bit) value) and name it DisableSearchBoxSuggestions.


Create the keys

If there is no key named Explorer right-click and create one. Then select it and on the right add a new DWORD value (right-click > New) and name it DisableSearchBoxSuggestions. Double-click it and set its value data to 1.


You're done!

Now all you have to do is restart Explorer.exe .
