How to disable web search results in the start menu

Windows 10 1809 and up

Written on February 11th, 2022


Open regedit

Press Windows + R, type regedit and hit enter. Accept the UAC prompt that shows up.
Navigate to HKEY_CURRENT_USER\SOFTWARE\Policies\Microsoft\Windows\.

Create the keys

If there is no key named Explorer right-click and create one.


Select it and on the right add a new DWORD value (right-click > New) and name it DisableSearchBoxSuggestions. Double-click it and set its value data to 1.


You're done!

Now all you have to do is restart Explorer.exe .
